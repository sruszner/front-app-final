FRONTEND APP final en REACT

PWA 2022-
Sebastian Ruszner

Pagina con 6 secciones navegables incluso desde el footer
Form de suscripción desde el footer
Desde contactos ("/contact") podrá completar un formulario, el cual hará un POST a la api (Backend)
Desde vistas ("/view table") podrá ver la info ingresada desde el formulario anterior (GET a la API).
En vistas tendrás dos botones "Edit" y "Delete" con modales para editar y eliminar la info respectivamente mediante un POST a la API

Se utilizó MONGO ATLAS como base de datos 
Front deloyado en Netlify
